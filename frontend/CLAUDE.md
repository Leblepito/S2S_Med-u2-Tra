# Frontend — React + TypeScript + Vite

## Kurallar
- Functional components + hooks (class component yasak)
- TypeScript strict mode
- Tailwind CSS (inline className, ayrı CSS dosyası yok)
- State management: React hooks (useState, useReducer, useContext)
- Audio: Web Audio API + AudioWorkletNode (ScriptProcessorNode kullanma — deprecated)
- WebSocket: native WebSocket API (socket.io kullanma — overhead fazla)

## Komutlar
```bash
npm run dev       # Development server (port 3000)
npm run build     # Production build
npm run test      # Vitest
npm run lint      # ESLint
```
